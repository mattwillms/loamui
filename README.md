**Stack:** React 19 路 TypeScript 路 Vite 路 Tailwind CSS 路 shadcn/ui

## Quick Start

```bash
cp .env.example .env
npm install
npm run dev
```

App runs at `http://localhost:5173`

Requires [LoamBase](https://github.com/mattwillms/loambase) running locally.
